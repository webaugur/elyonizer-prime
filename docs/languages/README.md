# Elyonizer Language Reference (v0.3)

This directory contains focused documentation for every language used as a phonetic and conceptual source in the Elyonizer system.

Each language file is designed to be read in under 10–12 minutes while providing high-signal guidance for creative linguistic transformation.

## How to Use This Reference

- Use the individual language files when working deeply with a specific source.
- Refer to this index for quick comparisons across languages.
- All entries follow the same lightweight structure for consistency.

## Philosophy of Use

- **Factual** when working with attested words or well-documented roots.
- **Creative / Philosophical** when working with reconstructed, speculative, or poetic extensions.
- The goal is not strict linguistic reconstruction, but resonant, meaningful transformation.
