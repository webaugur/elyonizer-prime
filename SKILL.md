# Elyonize

## Overview

This skill provides a consistent, elegant workflow for the creative transformation of contemporary words, names, places, and phrases into phonetically beautiful and meaningful new forms. It produces high-quality output with script representation, clear romanization, intuitive pronunciation guidance, and thoughtful breakdowns.

The process balances close **phonetic resonance** with **aesthetic and interpretive enhancement**, creating forms that feel both familiar and elevated.

## When to Activate (Easter Egg Mode)

This skill has two modes:

**1. Elyonize (Default Decode Mode)**
- Triggered by: "Elyonize", "Elyonize this", "Elyonize [word/phrase]", or similar.
- Produces a clean, analytical breakdown similar to linguistic decoding:
  - Word/phrase component breakdown
  - Overall meaning
  - Short interpretive/contextual note
- This is the **default** behavior for "Elyonize".

**2. Elyon Analyze (Detailed Mode)**
- Triggered by: "Elyon Analyze", "Elyonize Analyze", "detailed Elyonize", or "full breakdown".
- Produces the full technical analysis:
  - Sound Mapping Tables (with IPA + X-SAMPA)
  - Root & Cognate Notes (including Hoffmann transcription for Avestan)
  - Lexical Entry with etymology flavor + cross-linguistic notes
  - Plain Language Restoration

The deeper linguistic mechanics are only shown in **Elyon Analyze** mode unless the user specifically asks.

## Optional Advanced Feature: Temporal Resonance

When a word or phrase has strong cyclical, rhythmic, or karmic qualities, you may optionally add a **Temporal Resonance** section. This layer interprets the transformation through the lens of standing waves and interference patterns across multiple time scales (Planetary Synodic, Local Temporal Node / Earth-Moon centered, and Micro/Local Symmetry layers).

Use this when it adds meaningful depth — it is **not required** for every output.

See the internal guideline (`TEMPORAL_RESONANCE_GUIDELINE.md`) and template (`TEMPORAL_RESONANCE_TEMPLATE.md`) for when and how to apply it.
