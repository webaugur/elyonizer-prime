# Changelog

## [0.2] - 2026-06-20

### Changed
- Major restructure of modes:
  - `Elyonize` is now the **default light Decode mode** (clean breakdown + meaning + interpretation)
  - `Elyon Analyze` triggers the **full technical mode** (tables, roots, Lexical Entry)
- Added **Welsh** and **Basque** to cross-linguistic phonetic notes
- Added **selective German** support (with warning about vocabulary size)
- Added **Plain Language Restoration** section at the end of Analyze mode

### Improved
- Cleaner default output style matching user expectations
- Better documentation and quick reference

## [0.1] - Initial internal version
- Original creative transformation system with full technical tables and Lexical Entries
